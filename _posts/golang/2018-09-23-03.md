## 类型转化

* 类型转化,v.(type) 需要转换的类型
```go
   type data map[string]interface{} 
   var sub  = data{
       "zhangsan":"张三",
       "lisi":"李四",
   }  
   var parent data
   parent["sub"] = sub
   for k, v := range parent["data"].(data) {
       fmt.Printf("%s,%s", k, v)
   }
```
* 类型断言

```
    switch i := v.(type) {

    }
```
