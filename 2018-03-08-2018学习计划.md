**dockerFile:**

dockerFile分为4部分，基础镜像、维护者信息、镜像操作指令、容器启动命令
    
    #基础镜像
    From baseImage
    
    #维护人信息
    MAINTAINER name email
    
    #镜像操作执行令
    RUN instruct
    
    #容器启动指令
    CMD instruct   
    

**docker打标签:**

    docker tag image username/repository:tag
    
    docker push username/repository:tag
